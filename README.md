# 📊 E-Commerce Analytics Dashboard

**Interactive data analytics dashboard** analyzing 100,000+ orders from the Brazilian Olist E-Commerce dataset.

🔗 **Live Demo:** [https://rakeshhathack2145.github.io/ecommerce-dashboard/](https://rakeshhathack2145.github.io/ecommerce-dashboard/)

---

## ✨ Features

✅ **Real-time KPI Metrics** — Revenue, orders, customers, AOV, repeat rate, ratings  
✅ **Interactive Charts** — Revenue trends, payment methods, categories, reviews  
✅ **Responsive Design** — Works perfectly on desktop, tablet, and mobile  
✅ **Data Insights** — Key findings and business intelligence extracted  
✅ **Professional UI** — Clean, modern, recruiter-friendly interface  

---

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3, JavaScript (ES6+)
- **Visualization:** Chart.js library
- **Data Source:** Brazilian E-Commerce Public Dataset by Olist
- **Hosting:** GitHub Pages
- **Design:** Responsive, mobile-first approach
